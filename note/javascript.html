学習ノート-JavaScript

変数について

  • 1文字目に使えるのはアルファベット、アンダースコア、ドル記号
  • 2文字目以降は数字も使える
  • 大文字と小文字は区別される

比較演算子について

  • イコール2つは等価演算子（型まで見ない）
  • イコール3つは厳密等価演算子（型まで見る）

⇒(0 == "")がtrueになってしまうなどの弊害があるので、基本的には厳密等価演算子を推奨。

JQueryについて

  • JQueryでは、セレクタ、メソッド、パラメタを用いて操作する
    例：$('#img_unit').html('hoge')
    • セレクタ(#img_unit)は.containerのように、CSSと同じ形式で選択が可能
  • セレクタが同じ場合、セレクタを省略してメソッドを連続させるメソッドチェーンが使用可能
